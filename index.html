Instructions

Write a method called async_wait, that takes any number of arguments, each of which is an asynchronous function that takes a callback as an argument to call when they are done.

async_wait should return an object with a method then(fn) which takes as a parameter a function to execute when the asychronous functions are done.

Example usage

    async_wait(
      function(callback) {
        // do async stuff, tell them when we are done
        callback();        
      },
      function(callback) {
        // do async stuff, tell them when we are done
        callback();        
      }
    ).then(
      function() { alert('all done!'); }
    );
